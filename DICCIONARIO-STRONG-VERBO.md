# Diccionario Strong Verbo

## Estado incluido en esta versión

- 8.674 entradas hebreas: H1–H8674.
- Obra fuente: James Strong, 1890, dominio público.
- Estructura, presentación, buscador y enlaces internos adaptados por Verbo.
- Enlace directo desde RVG-+ V al tocar códigos H.
- El texto inglés original se conserva para evitar copiar traducciones españolas ajenas.

## Próxima incorporación

La sección griega G1–G5624 se integrará dentro del mismo módulo cuando esté disponible el archivo público `StrongsGreek.zip`. Hasta entonces, los códigos griegos pueden continuar usando el módulo de respaldo instalado.
