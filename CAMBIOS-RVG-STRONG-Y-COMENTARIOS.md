# Verbo — RVG Strong e indicadores de comentarios

## RVG + Strong Verbo

- Nombre visible: **Reina-Valera Gómez 2004 + Strong Verbo**.
- Abreviatura: **RVG+ Verbo**.
- La RVG original permanece intacta como módulo separado.
- La alineación Strong continúa identificada internamente como integración automatizada auditada de Verbo.

## Indicadores de comentarios

- El lector ya no consulta únicamente el comentario activo para decidir si muestra un indicador.
- Se cargan los comentarios disponibles para el capítulo actual.
- Cada versículo muestra un acceso independiente por cada módulo que tenga contenido aplicable.
- El indicador utiliza la abreviatura del módulo: Matthew Henry, Diario Vivir, LBLA, JFB, Ireneo, etc.
- Al tocarlo, Verbo selecciona el módulo correspondiente, abre el panel Comentario y se desplaza hasta la nota relacionada.
- El selector del panel cambia entre módulos sin volver a descargar todas las Biblias del capítulo.
